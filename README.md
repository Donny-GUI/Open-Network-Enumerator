Dont pay for home network scanners or whatever.

# Getting Started

windows

```
winget install nmap
git clone https://github.com/Donny-GUI/Open-Network-Enumerator.git
cd Open-Network-Enumerator
python main.py
```

linux

```
sudo apt install nmap
git clone https://github.com/Donny-GUI/Open-Network-Enumerator.git
cd Open-Network-Enumerator
python3 main.py
```
